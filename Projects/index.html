iOS Med Help

CMPE 297 - Emergent Software Technologies

MedHelp (iOS) – An iOS application as a medical health assistant, that helps people in recognizing the object, text detection with speech, Virtual Chatbot, QR scanner.

MedHelp is an iOS health application that benefits the senior citizens. The application proposes with the detection of the text in the pillbox using a smartphone’s camera. The application provides the architecture and implementation of a mobile application which helps in sharing the medicine information, find near by hospitals, a personal virtual assistant as a chatbot to answer the queries of the patient. The elderly people are benefited with this application as it not only helps in identifying the pill that avoids human error, but also with the augmented reality added as an additional feature helps in better visualization of the recognized object.
